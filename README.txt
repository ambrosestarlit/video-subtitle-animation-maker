Video Subtitle Animation Maker

GitHub Pagesでそのまま公開できる、動画字幕・テロップ用の字幕アニメーション作成ブラウザアプリです。

ファイル構成
- index.html
- style.css
- app.js
- README.txt

主な機能
- 音声ファイル読み込み
- 字幕TXT読み込み
  - TXTの1行を1つの字幕ボタンとして追加
  - 空行は自動除外
- 音声再生位置に合わせた字幕入力
  - 音声再生中、任意のタイミングで字幕ボタンをクリックすると、その時刻にフレーズを配置
  - 入力済みボタンはグレーアウト
  - 入力済みボタンをもう一度クリックすると、そのフレーズの配置をキャンセル
- 日本語 / 英語 / 韓国語 UI切り替え
- 16:9 / 9:16 のキャンバス切り替え
- 任意フォントの読み込み
  - ttf / otf / woff / woff2 に対応
  - 読み込んだフォントはJSON保存・キャッシュ保存に含まれます
- フレーズごとの個別設定
  - 開始秒 / 終了秒
  - 位置X / 位置Y
  - 左寄せ / 中央寄せ / 右寄せ
  - フォント / フォントサイズ / 文字色 / 行間
  - 字間 / 字間移動
  - 文字PAN移動 / サイズ移動 / 回転移動
  - 通常表示
  - タイプライター表示
  - 拡大しながら表示
  - ジャンプしながらタイプライター表示
  - ジャンプしながら表示
  - 登場時と退場時だけジャンプ
  - タイプライター速度
  - 拡大表示の最小サイズ / 拡大表示の速度
  - ジャンプの大きさ / ジャンプの速さ
  - フェードイン / フェードアウト
  - 縁取りの色 / 太さ
  - ドロップシャドウの色 / ぼかし / X / Y
- 指定文字列ごとの装飾
  - 1フレーズ中の特定文字列だけ、フォント / 色 / サイズ / フチ / 影を変更可能
  - 複数の指定文字列はコンマ区切りで登録可能
- 基本設定
  - キャンバス比率、基本フォント、基本サイズ、初期表示秒数、初期アニメーション、初期文字揃え
  - 新規フレーズは基本設定または直前のフレーズ設定を引き継ぎ
  - 全フレーズへ基本設定を一括反映可能
- プレビュー背景
  - 背景色
  - 任意の背景画像
  - 画面いっぱい / 全体表示 / 引き伸ばし
  - 背景画像サイズ、X移動、Y移動の調整
  - 背景はプレビュー確認用で、透過PNG書き出しには含まれません
- 書き出し
  - 透過PNG連番ZIP書き出し
  - SRT字幕ファイル書き出し
  - FPS、開始秒、終了秒、ファイル名接頭辞を指定可能
- JSON保存 / 読み込み
- IndexedDBキャッシュ保存 / 復元
- フレーズ設定プリセット保存 / 適用

公開方法
1. このフォルダ内のファイルをGitHubリポジトリへアップロードします。
2. GitHub Pagesを有効化します。
3. index.html がトップページとして表示されます。

注意
- 透過PNGのZIP書き出しには JSZip CDN を使用しています。
- 長尺・高FPSで書き出すと生成枚数が多くなり、ブラウザのメモリを多く使います。
- 書き出し時、背景色と背景画像は含まれません。動画編集ソフト側で背景映像や画像と合成してください。
- SRTはUTF-8 BOM付きで書き出します。
