Video Subtitle Animation Maker

GitHub Pagesでそのまま公開できる、動画字幕アニメーション作成支援用ブラウザアプリです。
字幕TXTを読み込むと1行ごとに字幕ボタンが作成され、音声を再生しながらクリックするだけで、その時刻へ字幕を配置できます。
透過PNG連番ZIPとSRT字幕ファイルの書き出しに対応しています。

ファイル構成
- index.html
- style.css
- app.js
- README.txt

主な機能
- UI言語切り替え
  - 日本語 / English
- 音声ファイル読み込み
- 字幕TXT読み込み
  - TXTの1行を1つの字幕ボタンとして追加
  - 空行は自動除外
- 音声再生位置に合わせた字幕入力
  - 音声再生中、任意のタイミングで字幕ボタンをクリックすると、その時刻に字幕を配置
  - 入力済みボタンはグレーアウト
  - 入力済みボタンをもう一度クリックすると、その字幕の配置をキャンセル
- 字幕ごとの個別設定
  - 開始秒 / 終了秒
  - 位置X / 位置Y
  - 左寄せ / 中央寄せ / 右寄せ
  - フォント / フォントサイズ / 文字色 / 行間 / 字間
  - 字間移動
  - 文字PAN X / Y
  - 文字サイズ倍率 / サイズ移動
  - 文字回転 / 回転移動
  - 通常表示
  - タイプライター表示
  - 拡大しながら表示
  - ジャンプしながらタイプライター表示
  - ジャンプしながら表示
  - 登場時と退場時だけジャンプ
  - タイプライター速度
  - 拡大表示の最小サイズ
  - 拡大表示の速度
  - ジャンプの大きさ
  - ジャンプの速さ
  - フェードイン / フェードアウト
  - 縁取りの色 / 太さ
  - ドロップシャドウの色 / ぼかし / X / Y
- 基本設定
  - 基本フォント、基本サイズ、初期表示秒数、初期アニメーション、初期文字揃え
  - 最初の字幕には基本設定を反映
  - 2つ目以降の字幕は、選択中または直前に入力した字幕の設定を引き継ぎ
- プレビュー背景
  - 背景色
  - 任意の背景画像
  - 画面いっぱい / 全体表示 / 引き伸ばし
  - 背景画像サイズのスライダー調整
  - 背景画像移動X / Y
  - 背景はプレビュー確認用で、透過PNG書き出しには含まれません
- 透過PNG連番ZIP書き出し
  - FPS、開始秒、終了秒、ファイル名接頭辞を指定可能
- SRT字幕ファイル書き出し
  - 入力済み字幕の開始秒 / 終了秒 / 本文から .srt を生成
- JSON保存 / 読み込み
- IndexedDBキャッシュ保存 / 復元
- 字幕設定プリセット
  - 選択中字幕の見た目・位置・アニメーション設定を名前付きで登録
  - 字幕本文、開始秒、終了秒は含めず、別字幕へ再利用可能
  - ブラウザのローカルストレージとJSON保存データに保持

公開方法
1. このフォルダ内のファイルをGitHubリポジトリへアップロードします。
2. GitHub Pagesを有効化します。
3. index.html がトップページとして表示されます。

使い方
1. 音声ファイルを読み込みます。
2. 字幕TXTを読み込みます。
3. 音声を再生しながら、表示したいタイミングで字幕ボタンをクリックします。
4. 入力済み字幕を選択し、開始秒・終了秒・位置・フォント・アニメーションを調整します。
5. 必要に応じて字幕設定プリセットへ登録します。
6. SRT字幕ファイル、または透過PNG連番ZIPを書き出します。
7. 透過PNG連番を書き出した場合は、動画編集ソフトで背景映像や画像と合成します。

注意
- 透過PNGのZIP書き出しには JSZip CDN を使用しています。
- 長尺・高FPSで書き出すと生成枚数が多くなり、ブラウザのメモリを多く使います。
- 書き出し時、背景色と背景画像は含まれません。動画編集ソフト側で背景映像や画像と合成してください。
- ローカルで直接開くより、GitHub PagesなどHTTP環境での利用を推奨します。
